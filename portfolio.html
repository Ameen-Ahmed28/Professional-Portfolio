DocLink Advisor using LLM

Developed an AI tool that processes a PDF or up to 3 URLs, stores the content in a FAISS vector database, and answers user questions using LLMs via LangChain. The model avoids hallucination by responding only when relevant data is found. Includes a summarization feature and a Streamlit-based user interface.

LLM RAG Financial Chatbot

Developed a stock analysis chatbot using Streamlit that integrates LLM-based financial advisory. The system fetches the latest news articles using Google Search, performs sentiment analysis using TextBlob, and generates actionable investment recommendations via Groq’s Llama3-based models. Real-time stock data is retrieved using the Finnhub API, and visualized with interactive Plotly charts.
